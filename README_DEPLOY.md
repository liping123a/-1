# ShopNS 腾讯云宝塔面板部署指南

既然您使用了宝塔面板，部署过程会变得更加简单。请按照以下步骤操作：

## 1. 安装 Node.js 和 PM2 (通过宝塔面板)

1.  登录宝塔面板，点击左侧菜单的 **“软件商店”**。
2.  在搜索框输入 **“Node.js”**。
3.  找到 **“Node.js版本管理器”** 并点击安装。
4.  安装完成后，点击该插件的 **“设置”**：
    *   在“版本列表”中，选择一个 **v18.x** 或更高版本点击安装。
    *   安装完成后，点击右侧的 **“命令行版本”**，选择您刚安装的版本，点击“确定”。
    *   PM2 通常会随版本管理器自动安装，您可以在“模块管理”中确认。

## 2. 安装 Puppeteer 系统依赖 (通过宝塔 SSH)

Puppeteer 运行需要一些 Linux 系统底层库，必须通过命令行安装：

1.  点击宝塔面板左侧菜单的 **“终端”**（或者使用您截图中的“宝塔SSH终端”）。
2.  如果是第一次使用，可能需要输入服务器的 root 密码登录。
3.  在终端中**依次**复制并运行以下命令（如果提示输入 Y/N，请输入 Y 并回车）：

```bash
sudo apt-get update
```

```bash
sudo apt-get install -y libnss3 libatk1.0-0 libatk-bridge2.0-0 libcups2 libdrm2 libxkbcommon0 libxcomposite1 libxdamage1 libxext6 libxfixes3 libxrandr2 libgbm1 libasound2
```

## 3. 上传项目代码

1.  点击宝塔左侧的 **“文件”**。
2.  进入 `/www/wwwroot/` 目录。
3.  点击 **“上传”**，将您的项目压缩包上传并解压到此目录下（例如解压到 `/www/wwwroot/shopns`）。

## 4. 配置数据库

1.  点击宝塔左侧的 **“数据库”** -> **“添加数据库”**。
2.  数据库名设为 `shopns_db`，设置好用户名和密码。
3.  在项目根目录下找到 `.env.example`，重命名为 `.env`，并修改其中的 `DB_USER` 和 `DB_PASSWORD` 为您刚设置的值。

## 5. 添加 Node 项目 (宝塔自动化部署)

1.  点击宝塔左侧菜单的 **“网站”** -> **“Node项目”**。
2.  点击 **“添加Node项目”**：
    *   **项目目录**: 选择您的项目文件夹（如 `/www/wwwroot/shopns`）。
    *   **项目名称**: shopns
    *   **启动选项**: 选择 `npm`。
    *   **运行脚本**: 输入 `start`。
    *   **项目端口**: 3000。
    *   **运行用户**: root (为了 Puppeteer 权限建议先用 root)。
3.  点击确定。宝塔会自动运行 `npm install` 并启动项目。

## 6. 配置外网访问 (反向代理)

1.  在“Node项目”列表中，找到刚才创建的项目。
2.  点击右侧的 **“映射”**，输入您的域名（如果没有域名，可以输入服务器 IP）。
3.  点击确定后，宝塔会自动为您配置好 Nginx 反向代理。

## 7. 腾讯云后台放行端口

别忘了在腾讯云控制台的 **“防火墙”** 页面，添加规则放行 **80** 端口。

现在，您就可以通过 `http://您的服务器IP` 访问您的 ShopNS 系统了！
